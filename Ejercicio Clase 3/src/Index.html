¿Qué es HTML5?

HTML5 es la versión moderna del lenguaje que se utiliza para crear páginas web. Su principal ventaja es que permite organizar mejor el contenido mediante etiquetas que indican qué representa cada parte del sitio.

Gracias a estas nuevas etiquetas, el código resulta más claro y ordenado, lo que facilita su comprensión y mantenimiento.

Dato adicional

Usar etiquetas como header, nav, section, article y footer ayuda a darle una estructura lógica al sitio web.
