# Ejercicio 2 — Desarrollo Web (Semana 2)

Repositorio para la **Problemática Semanal 02**: réplica en HTML semántico de una noticia de prensa online y práctica de flujo con Git y GitHub.

## Contenido

- `noticia.html`: artículo estructurado con etiquetas semánticas (`header`, `nav`, `main`, `article`, `section`, `aside`, `footer`, `blockquote`, `time`, etc.); sin imágenes respecto de la nota en línea.

## Noticia de referencia

Artículo de [*LA NACION*](https://www.lanacion.com.ar/sociedad/casi-el-19-de-los-argentinos-recurre-a-medicacion-para-enfrentar-el-malestar-emocional-nid13042026/) (Sociedad, 13/04/2026): **1 de cada 5 argentinos recurre a medicación para enfrentar el malestar psicológico**, por Sarah Mota. La réplica en HTML omite las imágenes de la nota original.

## Cómo ver el HTML

Abre `noticia.html` en el navegador (doble clic o “Abrir con…”).
